#### Curl

curl -G "baidu.com" --data-urlencode "paraml=valuel&param2=中文" -H 'Content-type:application/json'

- -G： GET 请求 默认POST
- --data-urlencode： 处理中文，空格等
- -H: 请求头
- -d POST 请求，后面跟请求体


curl -XPUT 'localhost:8080/index' -H 'Content-type:application/json' -d'

}
